# Real Teeth 채용과제

## 프로젝트 실행 방법

### 설치

```bash
pnpm install
```

### 개발 서버 실행

```bash
# 개발 모드
pnpm dev:dev

# 프로덕션 모드
pnpm dev:prod
```

### 빌드

```bash
# 개발 빌드
pnpm build:dev

# 프로덕션 빌드
pnpm build:prod
```

### 테스트

```bash
# 테스트 실행
pnpm test

# 테스트 감시 모드
pnpm test:watch

# 커버리지
pnpm test:coverage
```

## 구현한 기능에 대한 설명

### Feature/A-1-프로젝트기본세팅
**주요 작업 / 변경사항:**
- **React 19** - UI 라이브러리
- **Vite 7** - 빌드 도구
- **TypeScript** - 타입 안정성
- **Tailwind CSS 4, YD-UI** - 스타일링
- **React Query** - 서버 상태 관리
- **React Router** - 라우팅
- **Vitest** - 테스팅
- **Zustand** - 클라이언트 상태 관리
- **ESLint + Prettier** - 코드 품질 관리
- **Husky** - Git 훅 관리
- **FSD** - 프로젝트 아키텍처
- **스쿼시 머지**- GIT 워크플로우

**사용한 기술 및 이유:**

- 과제요구사항에 부합하면서 과제를 시간안에 처리하기 위해 익숙한 기술을 선택하였습니다.
